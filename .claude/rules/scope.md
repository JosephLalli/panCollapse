# Scope and development gates

- Phase 0 is research and architecture only.
- Phase 1 defines executable behavioral fixtures.
- Phase 2 is one thin vertical slice.
- Stop at every named development gate in `.agent-workspace/GATES.md`.
- Autonomous loops are allowed only for bounded, machine-verifiable tasks inside an
  approved phase.
- A judgment-sensitive ambiguity is a valid stopping condition; record it rather than
  guessing.
